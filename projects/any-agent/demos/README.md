# any-agent Demos

This directory is for stable demos worth keeping.

Current demos:

- `claude-agent-sdk-playground`: a runnable Node.js playground for `query()`, custom MCP tools, session flow, and minimal agent definitions

Possible future demos:

- a minimal Claude Code marketplace repo
- a minimal plugin with one skill
- an example showing how `any-agent` complements the official Agent SDK

Only keep demos here once they are understandable without extra chat context.
